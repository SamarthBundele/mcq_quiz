# MCQ Practice Website

A simple, clean web application to practice multiple-choice questions from your Excel file.

## Features

- Navigate through 318 MCQs one by one
- Instant feedback on your answers
- Track your score and progress
- Review correct answers
- Clean, responsive design

## How to Use

1. Open `index.html` in your web browser
2. Answer each question by clicking on an option
3. Use Next/Previous buttons to navigate
4. View your final score at the end

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - Quiz logic and interactivity
- `mcqs.json` - Questions data (converted from Excel)
- `convert_excel.py` - Script to convert Excel to JSON

## Running Locally

Simply double-click `index.html` or open it in any modern web browser.
